세포 호흡이란? 🌱

우리 몸의 세포가 포도당(음식)을 태워서 에너지(ATP)를 만드는 과정이에요.

포도당 + 산소(O₂)  →  이산화 탄소(CO₂) + 물(H₂O) + 에너지(ATP)
🧑‍🏫 세포 호흡은 3단계로 나눠서 일어나요.
① 해당 과정 → ② 피루브산 산화 · TCA 회로 → ③ 산화적 인산화(전자전달계)
각 단계를 하나씩 클릭하며 따라가 봅시다!

① 해당 과정

장소: 세포질
포도당 → 피루브산

② TCA 회로

장소: 미토콘드리아 기질
탄소를 CO₂로 배출

③ 전자전달계

장소: 미토콘드리아 내막
ATP 대량 생산

🔬 세포 구조 — 어디서 일어날까?

그림에서 세포질과 미토콘드리아를 클릭해 각 단계가 일어나는 장소를 확인해요.

👆 세포질 또는 미토콘드리아를 클릭해 보세요!
📍 세포질

🍬 해당 과정

포도당(탄소 6개)이 반으로 쪼개져 피루브산(탄소 3개) 2개가 돼요. 그림을 클릭할 때마다 한 단계씩 진행돼요.

🍬 포도당은 탄소 6개짜리 큰 분자예요. 먼저 ATP 2개를 써서 포도당을 활성화시켜요. 아래 버튼을 눌러 시작!
📍 미토콘드리아 기질

🔄 피루브산 산화 & TCA 회로

피루브산이 미토콘드리아로 들어가 탄소를 CO₂로 하나씩 내보내며 NADH·FADH₂·ATP를 만들어요.

🚪 피루브산(C₃)이 미토콘드리아 기질로 들어왔어요. 버튼을 눌러 반응을 하나씩 진행해 봅시다!
📍 미토콘드리아 내막

⚡ 산화적 인산화 (전자전달계)

지금까지 모은 NADH와 FADH₂가 전자를 전달해 ATP를 대량으로 만들어요.

⚡ NADH와 FADH₂가 준비됐어요! 버튼을 누르면 전자가 복합체를 지나며 H⁺을 퍼올리고, ATP 합성 효소를 돌려 ATP를 만들어요.

📊 정리 — 포도당 1개로 얼마나?

세 단계에서 만들어진 에너지를 모두 합해 볼까요?

단계장소ATP(직접)NADHFADH₂CO₂
① 해당 과정세포질2200
② 피루브산 산화기질0202
③ TCA 회로기질2624
합계—4 ATP10 NADH2 FADH₂6 CO₂
💡 NADH·FADH₂는 전자전달계에서 ATP로 바뀌어요. (NADH 1개 ≈ 2.5 ATP, FADH₂ 1개 ≈ 1.5 ATP)
모두 합하면 포도당 1개당 약 30~32 ATP가 만들어집니다!
C₆H₁₂O₆ + 6O₂ → 6CO₂ + 6H₂O + 에너지(약 32 ATP)

🍶 산소가 없으면? — 발효

산소가 없으면 전자전달계를 못 써요. 대신 발효로 해당 과정만 계속 돌려 적은 ATP(2개)를 얻어요.

알코올 발효

피루브산 → 에탄올 + CO₂
ATP 2개 생성 (효모·빵·술)

젖산 발효

피루브산 → 젖산 (CO₂ 없음)
ATP 2개 생성 (근육·요구르트)
